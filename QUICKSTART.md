#  Быстрый старт

## За 5 минут до работающего решения

### Шаг 1: Проверка системы (30 сек)

```bash
cd /usr/src/forte
python setup_check.py
```

Этот скрипт проверит:
-  Версию Python
-  Установленные пакеты
-  Структуру проекта
-  Наличие данных

### Шаг 2: Установка зависимостей (2 мин)

Если нужно:

```bash
pip install -r requirements.txt
```

### Шаг 3: Размещение данных (1 мин)

Поместите CSV файлы в `data/`:
- `транзакции в Мобильном интернет Банкинге.csv`
- `поведенческие паттерны клиентов.csv`

### Шаг 4: Обучение модели (5-10 мин в зависимости от размера данных)

```bash
python train.py
```

Что происходит:
1. Загрузка и объединение данных
2. Feature engineering (создание признаков)
3. Обучение CatBoost модели
4. Подбор оптимального порога
5. Расчёт метрик и SHAP
6. Сохранение модели и отчётов

### Шаг 5: Запуск веб-интерфейса (30 сек)

```bash
streamlit run app.py
```

Откроется браузер: http://localhost:8501

1. Используйте **API** (`api.py`)

## Помощь

**Ошибка при загрузке данных?**
- Проверьте кодировку (должна быть cp1251)
- Проверьте разделитель (должен быть `;`)

**Долго обучается?**
- Уменьшите `iterations` в `train.py` (строка с `FraudDetectionModel`)
- Используйте sample данных для быстрого теста

**SHAP не работает?**
- Убедитесь, что модель обучена
- Проверьте версию shap: `pip install shap --upgrade`
